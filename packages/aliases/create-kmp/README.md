# create-kmp

Official alias for [`create-cmp-cli`](https://www.npmjs.com/package/create-cmp-cli) — the AI
delivery harness for Kotlin/Compose Multiplatform. It scaffolds a green-building Android + iOS
app with a built-in, spec-driven verify lane. This package contains no logic of its own; it
delegates straight to `create-cmp-cli`.

```bash
npm create kmp@latest my-app
# or: npx create-kmp my-app
```

Docs, flags, and issues: https://github.com/kvdm-co-pilot/create-cmp
